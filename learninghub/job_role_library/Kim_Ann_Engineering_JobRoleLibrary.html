Replaces the Monday-morning Excel scramble with one company-wide view of the KPIs that matter, plus a wins / risks / decisions area that compiles itself into a meeting agenda.

Build Prompt
Build a General Manager's Executive Cockpit for the General Manager of Kim Ann Engineering, a Singapore industrial, electronics and oil-&-gas equipment distributor. Top section: 8 KPI cards displayed as large readable numbers with RAG (Red/Amber/Green) colour coding — Revenue vs Target, Gross Margin %, On-Time Delivery %, Cash Position (SGD), Order Backlog (SGD), Open Complaints, Headcount, and Overdue Receivables — each showing current value, target, and a small variance indicator. Middle: a Departmental Scorecard table (Sales, Operations, Finance, Service, HR) with a RAG status and one headline metric each. Below: a meeting prep area with three text fields — this week's wins, risks, and decisions needed — plus a "Generate agenda" button that concatenates them into a clean agenda text block for Outlook or Teams, and a Decision Log table. Bottom: a links panel with placeholder buttons for Sales, Operations, Finance, and HR dashboards (use [square brackets] for URLs). Polished executive aesthetic — navy with amber accents, generous whitespace, large typography. Modern, sleek, professional and corporate. Pure HTML, CSS, and vanilla JS with localStorage persistence. At the end, show the full, complete HTML source code in the code console/output as the result so I can copy it — don't only show a rendered preview.

💡 Prompt Tips

To get the best results from any of these prompts:

  • Paste the entire prompt into your AI tool in one go — don't trim it. The last line asks the AI to show the full HTML code in the console so you can copy it straight out.
  • After the first build, ask for iterations: "Add a dark mode toggle" or "Replace seed data with my actual customer list."
  • If the output feels generic, add context at the end: "Use terminology familiar to Singapore industrial and electronics trading."
  • Ask for a second file if needed: "Now give me a simple backend in Node/Express for this page."
  • Keep the unified framing in mind: each page should replace 3 common Excel files and 2 recurring emails in your job.
