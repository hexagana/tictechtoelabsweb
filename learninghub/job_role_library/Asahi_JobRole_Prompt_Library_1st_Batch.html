A curated set of webpage concepts aligned to the participant roles attending the Asahi Kasei Plastics 2-Day SFDW programme — each role gets copy-paste prompts that produce working internal tools, each replacing 3 Excel files and 2 recurring emails in the daily job across an engineering-plastics manufacturing operation.

How to use this guide

Find the section matching your job function, then copy any build prompt straight into an AI tool — Microsoft Copilot, ChatGPT, Claude, or Gemini. Each function now offers several tools — pick the one that speeds up the job you care about most. Every prompt is self-contained and produces a single HTML page with embedded CSS and vanilla JavaScript that you can preview in any browser and adapt to your team's actual data, and every prompt ends with an instruction to show the full HTML source code in the code console so you can copy it directly. Some roles are shared by more than one participant (e.g. three Senior Managers, two Engineers, two Assistant General Managers) — you all draw from the same section. The framing throughout is the same: "Build a browser-based, personal productivity cockpit that replaces 3 common Excel files and 2 recurring emails in your job."
