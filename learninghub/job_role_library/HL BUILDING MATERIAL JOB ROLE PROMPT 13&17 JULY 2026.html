How to use this guide

Find the section for your role, then copy any build prompt straight into any AI tool — Microsoft Copilot, ChatGPT, Claude, or Gemini. Each prompt is self-contained and produces a single HTML page with embedded CSS and vanilla JavaScript that you can preview in any browser and adapt to your team's real data. The framing is the same across every card: build a browser-based tool that replaces 3 common Excel files and 2 recurring emails in your job. Four ideas are offered per role — start with whichever solves your most annoying manual task first.

💡 Prompt Tips

To get the best results from any of these prompts:

  • Paste the entire prompt into your AI tool in one go — don't trim it.
  • After the first build, ask for iterations: "Add a dark mode toggle" or "Replace the seed data with my actual list."
  • If the output feels generic, add context at the end: "Use terminology familiar to a Singapore building-materials business."
  • Want it saved between visits? The prompts already ask for localStorage — just reopen the same file in the same browser.
  • Ask for a second file if needed: "Now give me a simple backend in Node/Express for this page."
